\# 🚀 CRM Pro - Modern CRM System



A full-featured Customer Relationship Management system built with PHP, Supabase, and Bootstrap 5.



!\[CRM Dashboard](screenshot.png)



\## ✨ Features



\- 📊 \*\*Dashboard\*\* - Real-time statistics and overview

\- 🏢 \*\*Companies Management\*\* - CRUD operations

\- 👥 \*\*Contacts Management\*\* - Linked to companies

\- 💼 \*\*Deals Pipeline\*\* - Kanban board view

\- ✅ \*\*Activities Tracking\*\* - Calendar and list views

\- 🎫 \*\*Support Tickets\*\* - Customer support system

\- 📈 \*\*Reports\*\* - 4 types with CSV/PDF export

\- 🔔 \*\*Notifications\*\* - Real-time updates

\- 👤 \*\*User Management\*\* - Role-based access control



\## 🔐 User Roles



| Role | Permissions |

|------|------------|

| Super Admin | Full access, user management |

| Admin | Manage companies, deals, reports |

| Company User | Limited to company data |



\## 🛠️ Tech Stack



\- \*\*Backend:\*\* PHP 7.0+

\- \*\*Database:\*\* Supabase (PostgreSQL)

\- \*\*Frontend:\*\* Bootstrap 5, Modern CSS

\- \*\*Icons:\*\* Font Awesome 6

\- \*\*Auth:\*\* Custom session-based



\## 📋 Requirements



\- XAMPP (Apache + PHP 7.0+)

\- Supabase Account

\- Web Browser
